## Main Features

- Based on Vite and [vite-plugin-ssr](https://vite-plugin-ssr.com/)
- Uses React 18, Suspense SSR Streaming enabled by default
- Deep integration with Relay (Supports render-as-you-fetch)
- Easy to use
